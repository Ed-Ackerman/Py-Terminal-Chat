Install the following packages before running, 

Installation commands

Run these commands in the terminal directory of the client folder

pip install pyfiglet
pip install socket
pip install threading
pip install rich
pip install msvcrt
pip install keyboard

After package installation open cmd and navigate to the client folder and run these commands in order;

py pantheon.py or python pantheon.py
py client.py or python client.py

Afer running py "client.py" enter your user name and send a message
